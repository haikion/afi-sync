Title:
- No Chain

Author:
- LuzifR

Description:
- This addon removes the red/yellow chain which appears on the screen when the connection quality is bad or the desync is high. I decided to make this addon for movie-makers because the chain does disturb in the middle of the screen.

Required Addons:
-

Signed:
- Yes, serverkey included

Installation:
- Extract the file(s) to your ArmA3 directory. Now you can use a following parameter to start ArmA3 with this mod: -mod=(mod folder).
